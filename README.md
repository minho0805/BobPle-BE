# 🍽️ BobPle-BE (밥플 - 백엔드)


---

## 🚀 주요 기능

- **회원 관리**
  - 회원가입 / 로그인 (JWT 인증)
  - 프로필 조회 및 수정

- **이벤트(밥약)**
  - 식사 약속(밥약) 생성 / 조회 / 수정 / 취소
  - 밥약 참가 신청 및 취소

- **리뷰**
  - 밥약 후 상대 평가
  - 받은 리뷰 조회

---

## 🛠 기술 스택

- **Backend**: Node.js, Express  
- **DB**: PostgreSQL, Prisma ORM  
- **Auth**: JWT, Firebase  
- **Docs**: Swagger(OpenAPI)  
- **Infra**: Docker, GitHub Actions (CI/CD)
